# First Contact with Three.js

Early experiments following the [Three.js documentation](https://threejs.org/docs/) — ended up going beyond the basics and making rotating cubes with video textures.

## What's in here

- `main.js` — Three spinning cubes with video textures playing against a background image
- `test2.js` — Additional Three.js experiments

## Stack

- [Three.js](https://threejs.org/)
- [Vite](https://vitejs.dev/)
- Vanilla JavaScript
